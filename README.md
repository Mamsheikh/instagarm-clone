- [x] Login with Google
- [x] Like / Unlike a photo

- [x] See user profile
- [x] See MY profile
- [x] See the single post
- [x] Upload a photo to cloudinary
- [x] Edit the caption (Delete)
- [x] See the feed
- [] Send private Message
- [] See rooms
- [] See room
